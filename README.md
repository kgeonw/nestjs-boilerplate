## Description

Build by [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy
Azure Container Registry
```
${ REGISTRY_LOGIN_SERVER }/${ ARCHIVE_NAME }:${ RELEASE_VERSION }
```

URL
```
${ dns-name-label }.${ location }.azurecontainer.io
```
